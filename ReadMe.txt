Dependancy
- Python 3
- Tkinter (for Python 3)
- py2exe (optional) - for setup.py (creates a exe)

How to run
1. Double click  Main.py
2. Copy and Paste notes (must follow convention, see example.txt)
3. Click "done"
4. use "<" and ">" to cycle left and right throught the different cards
5. Click "show" to flip a card (show definition)
